# Do Fish Have Ears? 
After a sacred text feel in the ocean, one little fish become conscious. There is always one immortal fish.

This fish has wandered for centuries, but finally dediced that enough is enough... 

He will become RADICAL!!

## Controls: 
<p> Turning: A and D</p>

<p> Moving: W and S</p>

<p> Flipping: Turning and Moving become flipping in the air</p>

## Scoring:

    1.) Moving in the air.
    2.) Performing flips in the air.</p>

## Becoming Cool! 
As all self-respecting fish know, an arbitrary number makes you COOL!

Score more points and see yourself become BODACIOUS!!
